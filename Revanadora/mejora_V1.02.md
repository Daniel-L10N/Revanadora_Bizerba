## Sensor Inductivo

En caso de utilizar un sensor inductivo, este se colocará en el centro del equipo. Esto reducirá las probabilidades de que el brazo de carga pase sin ser detectado. El sensor detectará el paso del brazo cuando este se desplace frente a él.

#### Para usar el sensor inductivo, se recomienda emplear la función `millis()` en lugar de `delay()`. Esto permitirá gestionar el tiempo de manera más eficiente y evitará bloqueos en el programa.

#### para usar el sensor a un voltaje mas alto al del microcontrolador se puede usar un optoacoplador o un transistor NPN, preferentemente un optoacoplador para aislar el microcontrolador del voltaje mas alto.

## Codigo de colores: 
##### Azul: GND Negativo
##### Marón: VCC Positivo
##### Negro: Señal
